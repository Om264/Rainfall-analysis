## 🌍 Project Overview

**RainWatch** is a two-part water resources project built for hydrology students
and researchers. It provides:

1. **`rainfall_analysis.py`** — A Python script that generates a sample rainfall
   CSV dataset, reads it with pandas, calculates key hydrological statistics
   (total, average, per-station breakdown, areal mean), and prints a formatted
   terminal report.

2. **`rainfall_dashboard.html`** — A standalone browser dashboard that visualises
   global rainfall data with live city cards, a monthly comparison chart, a
   station summary table, and a searchable city database — all with light/dark
   theme support and four language options.

Both components are fully self-contained and require no server setup.

## ✨ Features

### Python Script (`rainfall_analysis.py`)

| Feature | Description |
|---------|-------------|
| CSV generation | Creates a 21-row sample dataset across 3 stations, 7 days |
| Data loading | Reads CSV using pandas with automatic date parsing |
| Basin statistics | Total, average, max, min rainfall; rainy vs dry day count |
| Per-station analysis | Grouped totals, averages, maximums, dry-day counts |
| Areal mean rainfall | Arithmetic average across stations per day |
| Wettest day detection | Automatically flags the highest-rainfall day |
| Terminal report | Formatted 3-section report (A, B, C) with ASCII bar chart |

### Web Dashboard (`rainfall_dashboard.html`)

| Feature | Description |
|---------|-------------|
| 🌧 Live city cards | 12 cities currently raining with intensity badges |
| 🔍 Search bar | Search 20+ cities for rainfall data with autocomplete |
| 📊 Monthly chart | 2024 vs 2023 bar chart powered by Chart.js |
| 📋 Station table | Summary table with inline bar visualisation |
| 🌙 Light / Dark mode | Full theme system with smooth CSS transitions |
| 🌐 4 languages | English, 中文, العربية, Français — RTL layout support |
| 🌧 Rain animation | 55 CSS-animated raindrops in the background |
| 📱 Responsive | Works on desktop, tablet, and mobile browsers |

## 📖 Usage Guide

### Python Analysis Script

#### Using your own data

Replace the auto-generated CSV with real field measurements. Your file must
have exactly these three columns:

```csv
date,rainfall_mm,location
2024-06-01,12.4,Station_A
2024-06-01,8.7,Station_B
2024-06-02,0.0,Station_A
```

Then update the `CSV_FILE` variable at the top of the script:

```python
CSV_FILE = "my_field_data.csv"
```

#### Saving the report to a text file

```bash
python rainfall_analysis.py > rainfall_report.txt
```

#### What each section of the report means

| Section | Contents |
|---------|----------|
| A — Overall Basin Statistics | Total, average, max/min readings; rainy vs dry days; wettest day |
| B — Per-Station Breakdown | Individual station totals, averages, and dry-day counts |
| C — Daily Areal Mean | Day-by-day basin average with ASCII bar chart |

---

### Web Dashboard

#### Language switching

Click the **flag icon** (🇬🇧) in the top-left corner of the navbar to open
the language dropdown. Supported languages:

| Flag | Language | Layout direction |
|------|----------|-----------------|
| 🇬🇧 | English | Left-to-right |
| 🇨🇳 | 中文 (Chinese) | Left-to-right |
| 🇸🇦 | العربية (Arabic) | Right-to-left |
| 🇫🇷 | Français (French) | Left-to-right |

Every label, button, unit, and placeholder updates instantly when you switch.
Arabic activates a full RTL page layout automatically.

#### Theme switching

Click the **☀️ / 🌙 toggle** in the top-right corner:

| Theme | Background | Best for |
|-------|-----------|----------|
| Dark | Deep navy blue (#0b1120) | Night use, eye comfort |
| Light | Soft lavender-blue (#f0f6ff) | Daylight, presentations |

The bar chart palette and all UI colours update immediately.

#### Searching for a city

1. Click the search bar at the top of the page
2. Type any city name — autocomplete suggestions appear with rain status
3. A pulsing blue dot means the city is currently raining
4. Press **Enter** or click **Search**
5. A result panel slides in showing six data points:
   - Today's rainfall (mm)
   - Monthly total (mm)
   - Annual average (mm/yr)
   - Current humidity (%)
   - Weather condition
   - Wind speed (km/h)
6. Click **✕ Close** to dismiss

Cities in the built-in database:
```
Amsterdam   Bangkok     Beijing     Berlin      Cairo
Dubai       Dublin      Jakarta     Karachi     Lagos
London      Manaus      Mumbai      New York    Paris
São Paulo   Seattle     Singapore   Sydney      Tokyo
Vancouver
```

Any city not in the database returns realistically simulated data so
searches never fail.

## 📊 Dashboard Features

### Hero Statistics (top row)

| Card | What it shows |
|------|---------------|
| Cities Raining Now | Count of cities with active rainfall |
| Global Average | Mean daily rainfall (mm) across monitored stations |
| Highest Today | Peak single-location reading for today |

Each card shows: city name, country flag, rain intensity badge, current
rainfall in mm, temperature, wind speed, humidity, and a mini bar chart
comparing intensity to the highest current reading. Clicking a card opens
the full search result panel for that city.

### Monthly Comparison Chart

Side-by-side bar chart comparing 2024 vs 2023 monthly rainfall totals.
Hover over any bar for an exact mm reading. The colour scheme adapts to
the active theme.

### Station Summary Table

Summarises the five monitoring stations from the Python script:

| Column | Description |
|--------|-------------|
| Station | Gauging station identifier |
| Total (mm) | Cumulative rainfall over the study period |
| Avg/Day (mm) | Mean daily reading at this station |
| Days | Number of days with any recorded rainfall |

## 📂 Data Format

### Required CSV structure

```
date,rainfall_mm,location
2024-06-01,12.4,Station_A
2024-06-01,8.7,Station_B
2024-06-01,15.0,Station_C
2024-06-02,0.0,Station_A
```

| Column | Type | Required format | Notes |
|--------|------|-----------------|-------|
| `date` | string | `YYYY-MM-DD` | Must be ISO 8601 |
| `rainfall_mm` | float | `≥ 0.0` | Use `0.0` for dry days, not blank |
| `location` | string | Any label | Consistent spelling required |

### Rules and constraints

- File encoding must be **UTF-8**
- The header row (`date,rainfall_mm,location`) is mandatory
- Multiple stations on the same date is valid and expected
- Do not leave `rainfall_mm` blank — use `0.0` for dry days
- Dates must be in strict `YYYY-MM-DD` format (e.g. `2024-06-03`, not `6/3/2024`)

## ⚙️ Customisation

### Add more stations to the Python script

```python
sample_data = [
    # existing rows ...
    ["2024-06-01", 9.2,  "Station_D"],
    ["2024-06-02", 14.5, "Station_D"],
    # add as many rows as needed
]
```

### Add a city to the dashboard database

Find the `CITY_DB` object in `rainfall_dashboard.html` and append:

```javascript
"nairobi": {
    name: "Nairobi",
    country: "Kenya",
    today: 18,          // today's rainfall in mm
    monthly: 96,        // this month's total in mm
    annual: 890,        // long-term annual average in mm
    humidity: 76,       // relative humidity %
    condition: "Rain",  // weather description string
    wind: 15            // wind speed in km/h
},
```

### Add a new language

In `rainfall_dashboard.html`, find the `T` translations object and add your
language using the `en` entry as a template. Then add the flag button to the
language menu HTML and register the flag in the `LANG_FLAGS` map.

### Change the chart data

Find `CHART_2024` and `CHART_2023` arrays in `rainfall_dashboard.html` and
replace the 12 monthly values (Jan–Dec) with your own data:

```javascript
const CHART_2024 = [45, 52, 68, 82, 95, 114, 87, 91, 76, 58, 43, 61];
const CHART_2023 = [38, 49, 72, 78, 88, 108, 82, 95, 70, 52, 38, 55];
```

## 🛠 Troubleshooting

### Python script

| Problem | Likely cause | Solution |
|---------|-------------|----------|
| `ModuleNotFoundError: pandas` | Package not installed | `pip install -r requirements.txt` |
| `python: command not found` | Python not on PATH | Use `python3` instead |
| `PermissionError` creating CSV | No write permission | Run from your home or Documents folder |
| `NaN` values in output | Blank cells in CSV | Replace blank `rainfall_mm` cells with `0.0` |
| Date parsing fails | Wrong date format | Use strictly `YYYY-MM-DD` format |
| Script runs but no CSV | Wrong working directory | `cd` into the project folder first |

### Web dashboard

| Problem | Likely cause | Solution |
|---------|-------------|----------|
| Chart does not render | CDN blocked / no internet | Try a different network or browser |
| Arabic text displays incorrectly | Outdated browser | Update to Chrome 90+ or Firefox 88+ |
| Fonts look generic | Google Fonts blocked | Dashboard still works with fallback fonts |
| Search returns no result | City not in database | Simulated data is returned automatically |
| Page layout looks broken | Very old browser | Use a modern browser (released after 2021) |
| Theme toggle has no effect | JavaScript disabled | Enable JavaScript in browser settings |

## 📚 Glossary

| Term | Definition |
|------|------------|
| **Rainfall depth (mm)** | Volume of water per unit area. 1 mm = 1 litre per square metre of surface area. |
| **Areal mean rainfall** | Arithmetic average of readings from multiple stations on the same day, representing the basin-wide rainfall depth. |
| **Gauging station** | A fixed monitoring point that continuously records rainfall (and often flow, temperature, or other hydrological variables). |
| **Dry day** | A day on which no measurable rainfall was recorded (0.0 mm). |
| **Rainy day** | A day on which any measurable rainfall was recorded (> 0.0 mm). |
| **Drizzle** | Light rain with droplet diameter < 0.5 mm; intensity typically < 2.5 mm/hr. |
| **Heavy rain** | Rainfall with intensity > 7.6 mm/hr sustained over at least one hour. |
| **Areal rainfall estimation** | The process of estimating basin-wide rainfall from point (station) measurements. Common methods include arithmetic mean, Thiessen polygons, and isohyetal analysis. |
| **Flood alert** | An official warning issued when observed or forecast rainfall or river levels exceed defined safety thresholds. |
| **Basin** | A catchment area — the region of land that drains into a common river or water body. |
